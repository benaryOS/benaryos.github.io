# benaryOS

This is the benaryOS website. Great, you found it!

## What is benaryOS?

benaryOS is an opensource operating system aiming to provide a 100% GNU/Linux
compatible driver interface and some additional features making it possible to
replace a linux kernel with a benaryOS kernel.

## Why should I use benaryOS over GNU/Linux?

When benaryOS is finished, if it will ever be, it will be licensed under the MIT
license.

## What can I do here?

At the moment you can:

* use [git](http://github.benaryos.org)
* and our [bugtracker](http://github.benaryos.org/kernel/issues) (use this repo
only for kernel related bugs, other bugs should be put in their own
repositories).

## Contribution

If you want to contribute, just clone our repo (via git or http(s)) and upload
it to some git or fork it on GitHub and write an e-mail to
[pullme@benaryos.org](mailto:pullme@benaryos.org) or send is a pull request via
GitHub.
We will then do the rest and inform you about what we did.
